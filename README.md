Краткое описание:
Программа-утилита для подсчета количества страниц в документах, реализованная на Java 11

Ссылка для swagger UI: http://localhost:8080/swagger-ui/index.html#/

Доступные варианты JSON:
{
"url": "/.../...",
"docFormats": ["pdf"]
}

{
"url": "/.../...",
"docFormats": ["docx"]
}

{
"url": "/.../...",
"docFormats": ["pdf", "docx"]
}

{
"url": "/.../...",
"docFormats": [""]
}
